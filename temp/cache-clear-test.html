🧹 清除浏览器缓存步骤

方法一：强制刷新

• Windows/Linux: Ctrl + F5 或 Ctrl + Shift + R • Mac: Cmd + Shift + R

方法二：清除缓存

• Chrome: 按 F12 → Network 标签 → 右键点击刷新按钮 → "Empty Cache and Hard Reload" • 或者: 设置 → 隐私和安全 → 清除浏览数据 → 选择"缓存的图片和文件"

方法三：无痕模式测试

• 打开无痕/隐私浏览模式 • 访问网站测试功能
