后端异步架构

# Celery配置 - Broker: Redis (localhost:6379/2) - Backend: Redis (localhost:6379/1) - 任务队列: question_generation, ai_reference - 并发Worker: 2个 - 任务超时: 5分钟 - 重试机制: 3次，指数退避

  • 问题生成任务：处理AI问题生成，支持缓存检查

  • 错误处理：自动重试和错误报告

📋 部署说明

服务启动顺序

1. 启动Redis服务 brew services start redis 2. 启动Celery Worker cd backend source venv/bin/activate python start_celery_worker.py 3. 启动后端服务 cd backend source venv/bin/activate python run_complete.py 4. 启动前端服务 cd frontend npm run dev

配置检查

  • Redis连接：确保Redis服务正常运行
  • Celery Worker：检查worker是否正常启动
  • 任务队列：验证队列配置正确
